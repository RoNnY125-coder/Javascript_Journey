## ✅ Topics Covered

### 🔷 Basics
- Variables and Constants (`var`, `let`, `const`)
- Primitive Data Types (string, number, boolean, null, undefined, symbol)
- Type Conversion & Type Coercion
- Operators (arithmetic, comparison, logical, assignment)
- Conditional Statements (`if`, `else if`, `else`, `switch`)
- Loops (`for`, `while`, `do...while`)
- Functions (declaration, expression, arrow functions)
- Arrays & Array Methods
- Objects & Object Methods
- String Methods

### 🔷 DOM (Document Object Model)
- Selecting Elements (`getElementById`, `querySelector`, etc.)
- Modifying HTML content and attributes
- Changing CSS styles with JavaScript
- Event Listeners & Event Handling
- DOM Traversal (parent, child, sibling nodes)
- Creating and removing elements dynamically

### 🔷 Advanced
- Scope (global, local, block scope)
- Hoisting
- Closures
- Higher-Order Functions (`map`, `filter`, `reduce`)
- Promises & Asynchronous JavaScript
- `async` / `await`
- ES6+ Features (destructuring, spread/rest, template literals, modules)
